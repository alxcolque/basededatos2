Sintaxis

Una función al igual que los stored procedures, su código se guarda en la base de datos y tiene como función realizar un cálculo y devolver un valor.

Ejemplos

  • Ejemplo 1.
  • Contar todos los empleados del esquema HR 
    CREATE OR REPLACE FUNCTION GET_TOTAL_EMPLEADOS
RETURN NUMBER
IS
E_TOTAL NUMBER:=0;
BEGIN
SELECT COUNT(*) INTO E_TOTAL FROM EMPLOYEES;
RETURN E_TOTAL;
END;

-- Probando la función:

SELECT GET_TOTAL_EMPLEADOS FROM DUAL;
